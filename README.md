# My Portfolio

A beautiful, modern portfolio website showcasing professional work and projects.

## Overview

This is a responsive portfolio website built with HTML, CSS, and JavaScript. It features a elegant design with custom styling, smooth animations, and an interactive cursor effect.

## Features

- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Custom Cursor**: Interactive cursor with hover effects
- **Smooth Animations**: CSS transitions and smooth scroll behavior
- **Professional Typography**: Beautiful font combinations using Google Fonts
- **Modern Color Palette**: Carefully curated color scheme for a polished look
- **Interactive Elements**: Engaging hover effects and visual feedback

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/my-portfolio.git
```

2. Navigate to the project directory:
```bash
cd my-portfolio
```

3. Open `portfolio.html` in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" → your browser, or
   - Use a local server (recommended for development)

### Local Server (Optional)

For better development experience, use Python's built-in server:

```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

Then visit `http://localhost:8000` in your browser.

## Customization

### Colors

Modify the CSS variables in the `<style>` section of `portfolio.html`:

```css
:root {
    --navy: #1a2744;
    --terracotta: #b56a4a;
    --sage: #5a7a5e;
    /* ... other colors ... */
}
```

### Fonts

Fonts are imported from Google Fonts. Update the links in the `<head>` section to use different fonts.

### Content

Edit the HTML content in `portfolio.html` to add your own information, projects, and achievements.
